Item Handler Overview

The Item Handler provides a comprehensive system for managing and using items in your rotations. Whether you're dealing with consumables, equipment with on-use effects, or trinkets, the Item Handler offers a unified interface for item management.

What is the Item Handler?​

The Item Handler is Aurora's item management system that allows you to:

  • Create item objects with detailed properties and behaviors
  • Track equipped items automatically
  • Use items in rotations with built-in safety checks
  • Handle item cooldowns and availability
  • Manage both consumables and equipment

Key Features​

Automatic Equipment Tracking​

The system automatically scans your equipped items and maintains an armory of available items with on-use effects.

Smart Item Usage​

Built-in checks ensure items are only used when appropriate:

  • Line of sight validation
  • Cooldown management
  • Equipment status verification
  • Casting/channeling state checks

Flexible Item Creation​

Support for various item types:

  • Consumables - Potions, food, scrolls
  • Equipment - Trinkets, weapons, armor with on-use effects
  • Individual Items - Each item ID requires a separate item object

Getting Started​

The Item Handler integrates seamlessly with Aurora's rotation system. You can create items, add them to your armory, and use them in your rotation logic with simple, intuitive methods.

-- Basic item creation
local healthPotion = Aurora.ItemHandler.NewItem(33447)

-- Using an item
if healthPotion:usable(player) then
    healthPotion:use(player)
end
